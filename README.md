# 📊 Book Sales Analysis Dashboard

This project analyzes book sales data and provides an interactive dashboard.

## 🚀 Features

- Top 5 revenue days
- Unique users count
- Author set analysis
- Most popular author
- Top customer
- Daily revenue visualization
- Dataset switch (DATA1 / DATA2 / DATA3)
- Date filtering

## 🛠 Tech Stack

- Python
- Pandas
- Streamlit

## ▶️ Run locally

`bash
pip install -r requirements.txt
streamlit run app.py
